## 安装

- 安装服务 （需要先安装 NodeJS)

```bash
sudo npm install -g hello-dt
sudo hello-dt -i
```

- 打开 Hello-DT 应用扫描安装中出现的二维码

- 开启服务

```bash
sudo hello-dt --start
```

- 开机启动

```bash
sudo hello-dt -e
```
